+ KONFINGURASI
Pada dashboard menu kanan lakukan konfigurasi dahulu
Ubah pada menu locale dan locale pilih system default
Dan lakukan konfigurasi printer kasir yang kamu gunakan.
setelah itu restart program toko gratis,tutup dan buka lagi.

+ ADMIN
Lihat dashboard kanan pada menu admin dan pilih menu database
1. Buat database user dan hak aksesnya dahulu
2. Buat data Kategori group produk
3. Buat dan daftarkan produk jual mu
4. Masukan dan buat master pajak
5. Lakukan in out stok dan pembelian pada stok harian

+ TRANSAKSI
1. Masuk menu penjualan dan pilih kategori produk kemudian pilih produk untuk transaksi.
2. Selanjutnya untuk menghapus transaksi yang salah pilih fitur tanda stop (hapus transaksi) dan pilih backspace pada display penjualan untuk menghapus salah satu produk.
3. Kemudian klik tanda sama dengan (=) untuk melakukan pembayaran dan masukan pembayaran pelanggan.
4. Setelah transaksi jika menggunakan printer maka printer akan secara otomatis melakukan print
5. Penjualan support dengan barcode dan touchscreen mode.

+ EDIT PENJUALAN
Untuk melakukan edit penjualan transaksi caranya masukan no struk kasir kemudian lakukan edit sesuai kebutuhan.

+ CLOSE CASH
Fitur untuk melakukan tutup kasir bisanya dilakukan ketika toko tutup untuk melihat omset pendapatan harian mu.

+ LAPORAN
fitur untuk cek laporan laporan detail mulai dari stok hingga laporan keuangan.

+ KELUAR
Fitur untuk logout dan keluar dari program toko gratis.
